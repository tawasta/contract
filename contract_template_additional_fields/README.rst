====================================
Contract: Template Additional Fields
====================================

* Adds new fields to the contract template:

  * Invoice interval
  * Payment Terms
  * Responsible person

Configuration
=============
\-

Usage
=====
* Just open a contract template via Invoicing -> Configuration -> Contract Templates
  to see the new fields
